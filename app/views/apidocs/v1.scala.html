eval.gd API Version 1.

Want to develop applications around eval.gd? Check out our simple API. It's so simple, it's almost impossible to get wrong.

Evaluate Code

POST /evaluate

Request

{
  "api_version": 1,
  "language": "ruby",
  "code": "puts 2+2"
}

Response

{
  "stdout": "4",
  "stderr": "",
  "wall_time": "0.020"
}

GET /languages

Response

{
  "languages": [
    "apricot",

    "c",
    "c#",
    "c++",
    ...

}
