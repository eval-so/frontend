API Access

Here's what a typical request to the API looks like:

{
  "language": "php",
  "code": "<?php
              echo \"This is a test\";
           ?>"
}

Pretty straightforward, eh? You can, of course, tweak some more options. Check out our documentation for a full list of options, but for many languages, the above is all you need to do.

Command-Line Interface

Use the Command-Line Interface (CLI) to evaluate code right from your own computer. 

# Infer language based on file extension
$ breakpoint hello_world.rb
Hello, world!

# The exit code is that of the evaluation.
$ echo $?
0

# STDIN is sent along with the evaluation request, too.
$ echo '2' | breakpoint add_arg_plus_2.rb
4

# Get some more information about the request...
$ breakpoint --verbose demo.rb
