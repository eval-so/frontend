Sandboxed code evaluation platform.

Use Eval.so's API to compile and/or evaluate code safely and easily.

Why Eval.so?

Eval.so provides a free and open source web platform for you to evaluate code in many situations. Whether you want to make sure your code works on another computer, or you found some code online that you don't trust, simply run it on Eval.so, and have your answer in no time.

Where should I use it?

We'll leave that up to you! Eval.so prvodes an very simple JSON API. You can develop programs and websites that interact with the API, or take advantage of already-written programs that do. We also have an official command-line app that you can try out, which was written so that we have an application to use the API against, as we develop it.

How do I use it?

If you want to use a program that integrates with the API, simply download the program and use it. As of right now, the API doesn't require any form of authentication, so programs should "just work."
