Code evaluation in the cloud.

Eval.so provides an open source platform based on SELinux sandboxes, for evaluating code in the cloud.

Simple JSON API

POST /evaluate

Request

{
  "api_version": 1,
  "api_key": "35159d04-64f4-4948-b5c4-2a1f513e0cd9",
  "language": "ruby",
  "code": "puts 2+2"
}

Response

{
  "stdout": "4",
  "stderr": "",
  "wall_time": "0.020",
  "remaining_evaluations": 56
}

Official Command-line Utility

Eval.so's official command-line app allows you to trigger code evaluations and allows Eval.so administrators to control various aspects of the infrastructure, right from the terminal.

It's written in Go and open source, like all of Eval.so. It will soon be available as a package for most Linux distributions. 

$ cat hello-world-and-exit-with-123.rb | evalso --language=ruby
Hello, world!

$ echo $?
123
