Sandboxed code evaluation API.

Eval.so provides an open source platform based on SELinux sandboxes, for evaluating code on the web.

Simple JSON API

POST /evaluate

Request

{
  "language": "ruby",
  "code": "puts 2+2"
}

Response

{
  "stdout": "4",
  "stderr": "",
  "wallTime": "394",
  "exitCode": 0
}

Official Command-line Utility

Eval.so's official command-line app allows you to trigger code evaluations and allows Eval.so administrators to control various aspects of the infrastructure, right from the terminal.

It's written in Go and, like all of eval.so, it's open source. It will soon be available as a package for most Linux distributions. 

$ cat hello-world-and-exit-with-123.rb | evalso --language=ruby
Hello, world!

$ echo $?
123
